pug 语法

8月 10 2018 tutorial>grammar 5 分钟 读完 (约 824 字)

pug 是一个 JavaScript 的高性能模板引擎，被使用在 Node.js 和 浏览器 项目中。是由 Jade 重命名而来

参考
https://pugjs.org

符号与汉含义

符号 含义
. 表示之后的内容翻译为纯文本
: 表示嵌套关系，不需要换行
| 表示之后的内容翻译为纯文本，不识别 html 标签
// 注释，在编译出来的 html 中显示
//- 注释，在编译出来的 html 中不显示
- 后接 js 代码
#{变量} 转义插值
!{变量} 非转义插值

模板关键字

关键字 描述
block 占位符，通过其实现子模版的追加和替换
entends 继承，子模版继承父模板中的代码
appends 在子模版中向后追加内容
prepend 在子模版中向前追加内容
include 引入文件，并将其中的代码放到模板中
